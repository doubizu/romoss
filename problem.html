定制必看 常见问题

Q : 定制产品多久发货？
A : 个人定制发货时间是24小时，企业定制正常发货时间是48小时。我们互相根据亲们付款的先后顺序生产发出的。大促活动时间除外。（例如：双十一....）
Q : 定制效果怎么样？我想先看下效果可以吗？
A : 定制效果可以参考电脑设计图，实物是没有办法看到的，因为生产有很多流程，没有办法停下来--拍照的。
Q : 如果产品我不喜欢，可以退货吗？
A : 定制产品属于私人物品，刻上了属于自己的专属，我们没有办法二次销售，故不能退换货哦！
Q : 为什么生产出来的效果跟设计的时候有误差？
A : 我们是激光镭雕，生产出来的颜色跟网上看到的平面图会有少许误差，是很正常的哦！
Q : 我设计好图案并提交了，但是我想在更改一下内容可以吗？
A : 设计好的图案直接进入生产系统，我们是没有任何权限去改动的，如果亲们有改动建议联系我们的在线客服。重新设计，提供设计单号给我们。
Q : 你们的sense6移动电源容量是多大的，充满一次电大概需要多长时间？
A : 亲,您好，我们sense6是2000mAh,为sense6充满电用5V2.1A适配器充一次大概需要13小时（5V1A输入需26小时）
